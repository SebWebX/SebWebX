##  Proyectos Destacados

### Portafolio Web Profesional
Desarrollé y diseñe mi sitio web profesional, realizando lo siguiente:
• Elabore un portafolio con HTML5 semántico, CSS3 y JavaScript ES6+, aplicando metodología BEM para un código modular que facilita el mantenimiento y escalabilidad del proyecto.
• Implementé un sistema de diseño con CSS Custom Properties, tipografía fluida con clamp(), layouts con CSS Grid y Flexbox y animaciones con cubic-bezier personalizado.
• Desarrollé módulos JS independientes con scroll reveal via IntersectionObserver API y smooth scroll nativo, generando una experiencia de navegación fluida e intuitiva para el    usuario.
• Integré formulario con validación regex y consumo de API REST de Formspree, permitiendo que los visitantes contacten directamente sin abandonar el sitio.
• Apliqué flujo de trabajo profesional con Git, GitHub y deploy en GitHub Pages, garantizando entrega continua y control total sobre las versiones del proyecto  
- Link: https://sebwebx.github.io/mi-portafolio/
  
### App de Inventario (CRUD + LocalStorage)
Sistema de gestión de inventario del lado del cliente con renderizado dinámico del DOM, persistencia de datos con LocalStorage y cálculos automáticos del total de productos. Seguimiento de nombre, SKU, categoría, cantidad, stock mínimo y precio. Estilos con CSS Custom Properties para theming, CSS Grid y Flexbox para layout. Control de versiones con Git y GitHub.
