## ⭐ Proyectos Destacados
### 📦 App de Inventario (CRUD + LocalStorage)
Sistema de gestión de inventario del lado del cliente con renderizado dinámico del DOM, persistencia de datos con LocalStorage y cálculos automáticos del total de productos. Seguimiento de nombre, SKU, categoría, cantidad, stock mínimo y precio. Estilos con CSS Custom Properties para theming, CSS Grid y Flexbox para layout, completamente responsivo mediante media queries. Control de versiones con Git y GitHub.  
- Repo: https://github.com/SebWebX/App_Inventario
### 🌤️ App del Clima (API + UI)
Aplicación meteorológica que consume la API REST de WeatherAPI mediante Fetch API con encadenamiento de Promesas, renderizando clima actual y pronóstico de 4 días de forma dinámica parseando respuestas JSON al DOM. Manejo de errores asíncronos con `.catch()`, toggle claro/oscuro con CSS Custom Properties y persistencia del tema con `localStorage`. Incluye spinner animado con `@keyframes` CSS.  
- Repo: https://github.com/SebWebX/App_Clima
### 🔐 Generador de Contraseñas
Generador de contraseñas aleatorias y seguras con opciones configurables: longitud, mayúsculas, minúsculas, números y caracteres especiales. Lógica construida con JavaScript vanilla (ES6+), manipulación del DOM y retroalimentación visual clara al usuario. Incluye copia al portapapeles con un clic. Desarrollada con HTML5, CSS3 y JavaScript.  
- Repo: https://github.com/SebWebX/App_Contrase-a
